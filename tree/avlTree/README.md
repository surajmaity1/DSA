### AVL Treee

- calculate edge ( not node )
- It is a height balanced BST ( binary search tree)
- balance factor (bf) = height of left subtree (hl) - height of right subtree (hr)
- bf = hl - hr = {-1, 0, 1}
- bf = | hl - hr | <= 1 ( balanced )
- bf = | hl - hr | > 1 ( imbalanced )
